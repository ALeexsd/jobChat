# 📄 Быстрый гайд: Просмотр документов в чатах

## 🎯 Что это?

Теперь можно просматривать PDF и Word документы прямо в чатах без скачивания!

## 🚀 Как использовать

### Отправка документа

1. Откройте чат
2. Нажмите кнопку **"+"** в поле ввода
3. Выберите **"Документ"**
4. Выберите файл (PDF, Word, Excel, PowerPoint и др.)
5. Нажмите **"Отправить"**

### Просмотр документа

1. Кликните на прикрепленный документ в сообщении
2. Документ откроется в полноэкранном просмотрщике

### Навигация (для PDF)

- **◀️** - Предыдущая страница
- **▶️** - Следующая страница
- **⬇️** - Скачать документ
- **✖️** - Закрыть просмотрщик

## 📋 Поддерживаемые форматы

### С просмотром в браузере
- ✅ **PDF** (.pdf) - постраничный просмотр
- ✅ **Word** (.doc, .docx) - конвертация в HTML

### Только скачивание
- 📥 Excel (.xlsx, .xls)
- 📥 PowerPoint (.pptx, .ppt)
- 📥 Текстовые файлы (.txt)
- 📥 Другие форматы

## 💡 Советы

1. **Для больших PDF** - дождитесь полной загрузки
2. **Word документы** - форматирование сохраняется
3. **Темная тема** - работает и в просмотрщике
4. **Мобильные** - полностью адаптивно

## 🎨 Скриншоты функций

### Меню прикрепления
```
┌─────────────────┐
│ 🖼️ Изображение  │
│ 📄 Документ     │
└─────────────────┘
```

### Просмотрщик PDF
```
┌────────────────────────────────────┐
│ document.pdf          ◀️ 1/5 ▶️ ⬇️ ✖️│
├────────────────────────────────────┤
│                                    │
│         [PDF Content]              │
│                                    │
└────────────────────────────────────┘
```

### Просмотрщик Word
```
┌────────────────────────────────────┐
│ report.docx                   ⬇️ ✖️│
├────────────────────────────────────┤
│                                    │
│  # Заголовок                       │
│  Текст документа...                │
│                                    │
└────────────────────────────────────┘
```

## ⚡ Быстрые клавиши

- **Escape** - Закрыть просмотрщик
- **← →** - Навигация по страницам PDF (планируется)

## 🔧 Что делать если...

### Документ не открывается
1. Проверьте формат файла
2. Попробуйте скачать и открыть локально
3. Проверьте размер файла (рекомендуется < 10MB)

### Медленная загрузка
1. Дождитесь полной загрузки
2. Проверьте интернет-соединение
3. Для больших файлов используйте скачивание

### Неправильное форматирование Word
1. Сложное форматирование может отображаться некорректно
2. Скачайте оригинал для точного просмотра

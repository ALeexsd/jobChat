# Исправление форматов дат

## Проблема
API ожидает datetime в формате ISO 8601, но формы отправляли только даты.

## Исправления

### 1. CreateRouteModal.vue
- ✅ Изменен тип поля с `type="date"` на `type="datetime-local"`
- ✅ Поле теперь называется "Дата и время маршрута"
- ✅ Значение отправляется напрямую без преобразования

**Формат отправки:**
```json
{
  "date": "2024-01-01T10:00"
}
```

### 2. CreateTaskModal.vue
- ✅ Изменен тип поля дедлайна с `type="date"` на `type="datetime-local"`
- ✅ Значение отправляется напрямую без преобразования

**Формат отправки:**
```json
{
  "deadline": "2024-01-01T10:00"
}
```

### 3. CreateVacationModal.vue
- ✅ Оставлен `type="date"` (API ожидает только дату, не время)

**Формат отправки:**
```json
{
  "start_date": "2024-01-01",
  "end_date": "2024-01-10"
}
```

## Что теперь работает

1. **Маршруты** - можно указать дату и время начала маршрута
2. **Задачи** - можно указать точный дедлайн с временем
3. **Отпуска** - указываются только даты (без времени)

4. Создание заметки (без дат)
